
# NOVA MEGA STORE
A responsive, user-friendly, and interactive e-commerce website built using **HTML, CSS, and JavaScript**. This project demonstrates the core concepts of front-end web development by providing a realistic online shopping experience with product browsing, category collections, shopping cart management, login authentication, product search, and order confirmation.



## project overview
Nova Mega Store is a simple yet modern e-commerce application developed without any external frameworks. The website provides a clean user interface where users can securely log in, browse different product categories, view brand collections, add products to the shopping cart, manage their cart, and place orders successfully.

This project is designed for educational purposes and demonstrates how an online shopping platform works using only front-end technologies.
